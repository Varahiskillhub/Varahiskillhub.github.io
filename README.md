# Portfolio (GitHub Pages)

This is a lightweight, job-application-focused portfolio site.

## Deploy on GitHub Pages (recommended)
1. Create a repo named: `<your-username>.github.io`
2. Upload these files to the repo root (or push with git).
3. In GitHub: **Settings → Pages → Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: `main` / `(root)`
4. Your site will be live at: `https://<your-username>.github.io`

## Edit content
- `index.html` (main page)
- `pages/*` (case studies/projects)
- `assets/style.css` (styling)
- `assets/Bala_Donthamsetti_CV.pdf` (download link)
